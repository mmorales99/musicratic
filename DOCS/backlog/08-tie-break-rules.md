# Feature

Tie-break configuration

# Description

Implement the configured tie-break mode: downvote priority, upvote priority, or shop owner required.

# Acceptance Criteria

- The shop owner can configure the tie-break mode.
- Downvote priority resolves ties in favor of downvotes.
- Upvote priority resolves ties in favor of upvotes.
- Shop owner required forces the owner to resolve the tie.
